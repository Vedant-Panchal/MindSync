<h2>üßê Features</h2>

Here are some of the project's best features:

### Journaling

- AI-Powered Prompting ‚Äì Smart suggestions and reflective questions to help you write meaningfully
- Rich Text Editor ‚Äì Markdown + plain text support with smooth formatting experience
- Autosave ‚Äì Entries are saved in real-time to prevent data loss
- Journal History ‚Äì Access and browse past entries by date or search

### Insights & Summarization

- AI Summaries ‚Äì Get concise summaries of your entries using NLP
- Mood Detection ‚Äì Automatically detects mood or tone from your writing
- Streak Tracker ‚Äì Encourages consistent writing with streak and habit visualizations

### Authentication & User Management

- Email + OTP Authentication ‚Äì Simple sign-up and login with email verification
- Google OAuth ‚Äì Sign in using your Google account
- Session Persistence ‚Äì Cookie-based secure login with token refresh mechanism

### UI & UX

- Step-based Onboarding ‚Äì Smooth registration experience using multi-step forms
- Responsive Design ‚Äì Works seamlessly on mobile, tablet, and desktop

### Tools and Technologies

- FastAPI Backend ‚Äì Python-based backend with clean REST APIs
- PostgreSQL & Redis ‚Äì Robust data storage and caching
- Zustand for State Management ‚Äì Lightweight and scalable global state in frontend
- TanStack Router ‚Äì Type-safe, file-based routing
- React Query Auth Integration ‚Äì Secure token-based data fetching

## ‚öôÔ∏è Project Setup

#### üì¶ Prerequisites

Make sure you have the following installed before setting up the project:

- [`pnpm`](https://pnpm.io/installation) ‚Äì For frontend package management
- [`poetry`](https://python-poetry.org/docs/#installation) ‚Äì For backend Python dependency management
- Python 3.10+
- Node.js 18+
- Create a virtual environment for the backend (recommended)

---

### üñ•Ô∏è Frontend Setup

1. Create a `.env` file in the `frontend` directory:

   ```env
   VITE_API_URL=http://localhost:8000/
   ```
2. Navigate to the `frontend` directory:

   ```bash
   cd frontend
   ```

3. Install dependencies:

   ```bash
   pnpm install
   ```

4. Start the development server:

   ```bash
   pnpm dev
   ```


---

### Backend Setup

1. Create a `.env` file in the `backend` directory with the following format:

   ```env
   SUPABASE_URL=""
   SUPABASE_KEY=""
   JWT_SECRET=""
   JWT_ALGORITHM=""
   ACCESS_TOKEN_EXPIRE_MINS=""
   REFRESH_TOKEN_EXPIRE_DAYS=""
   RESEND_API_KEY=""
   ENCRYPTION_KEY=""
   UPSTASH_REDIS_URL=""  #https://upstash.com/
   UPSTASH_REDIS_PASSWORD=""
   OTP_EXPIRY_MINS=""
   GOOGLE_CLIENT_ID=""
   GOOGLE_CLIENT_SECRET=""
   GOOGLE_REDIRECT_URI="http://localhost:8000/auth/v1/google/callback"
   VECTOR_MODEL=" "  #Huggingface link https://huggingface.co/sentence-transformers/all-MiniLM-L6-v2"
   GEMINI_KEY=""
   ```

2. Navigate to the `backend` directory:

   ```bash
   cd backend
   ```

3. Create and activate a virtual environment (if not already):

   ```bash
   python -m venv .venv
   source .venv/bin/activate  # or .venv\Scripts\activate on Windows
   ```

4. Install dependencies using Poetry:

   ```bash
   poetry install
   ```

5. Run the backend server:
   ```bash
   poetry run uvicorn app.main:app --reload
   ```
